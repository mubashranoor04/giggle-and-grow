# Giggle and Grow

Giggle and Grow is an interactive educational mobile application developed using Flutter and Firebase. The application provides a fun and engaging learning experience for children through educational games, stories, drawing activities, alphabet and number learning, animal sounds, and shape recognition.

## Features

### Authentication

* User Sign Up
* User Login
* Password Reset
* Firebase Authentication

### Learning Modules

* Alphabet Learning
* Number Learning
* Animal Sounds
* Shape Recognition

### Games

* Memory Match Game
* Color Match Game
* Puzzle Activities

### Creative Activities

* Interactive Drawing Canvas
* Coloring Activities

### Stories

* Story Collection
* Interactive Story Reading

### Progress Tracking

* Score Management
* Reward System

## Architecture

This project follows Clean Architecture principles with separate layers for:

* Presentation Layer
* Domain Layer
* Data Layer

## Technologies Used

* Flutter
* Dart
* Firebase Authentication
* Cloud Firestore
* Provider State Management

## Installation

1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/giggle-and-grow.git
```

2. Install dependencies

```bash
flutter pub get
```

3. Run the application

```bash
flutter run
```
